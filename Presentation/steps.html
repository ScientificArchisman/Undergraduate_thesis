Subprocesses of Training a Machine Learning Model

Forward Propagation

Neural Network
The neural network takes in the input data and performs a series of calculations to generate a predicted output. This involves multiplying the input data by weights (which are initialized randomly) and applying activation functions.

Loss Calculation

Loss Function
The loss function measures the difference between the predicted output and the true output. Common loss functions include mean squared error, cross-entropy, and binary cross-entropy.

Backward Propagation

Gradient Calculation
The gradient of the loss function with respect to the model parameters is calculated using the chain rule of calculus. This involves computing the partial derivatives of the loss with respect to each parameter.

Parameter Update

Optimization Algorithm
The model parameters are updated using an optimization algorithm such as gradient descent. Gradient descent adjusts the parameters in the direction of the negative gradient of the loss function, with the goal of minimizing the loss.

Repeat

Looping
The previous steps (forward propagation, loss calculation, backward propagation, and parameter update) are repeated multiple times, with different batches of input data (in batch training) or individual samples of input data (in stochastic training). This process is known as an epoch, and the number of epochs is a hyperparameter that can be tuned to improve model performance.
